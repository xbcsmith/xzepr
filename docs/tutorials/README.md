# Tutorials

Welcome to the XZEPR tutorials section. These learning-oriented guides will help
you get started with XZEPR through hands-on practice.

## What are Tutorials?

Tutorials are step-by-step lessons designed to help you learn by doing. They
take you through a series of steps to complete a project, teaching you key
concepts and skills along the way.

**Characteristics of tutorials:**

- Learning-oriented
- Follow along and learn by doing
- Get you started and build confidence
- Teach through practical examples
- Have a clear beginning and end

## Available Tutorials

## Who Should Use Tutorials?

Tutorials are perfect if you:

- Are new to XZEPR and want to learn the basics
- Prefer hands-on learning by doing
- Want to understand core concepts through practice
- Need a guided introduction to the system

## What You'll Need

Before starting these tutorials, you should have:

- Basic command line knowledge
- Docker and Docker Compose installed
- A text editor or IDE
- Basic understanding of REST APIs (helpful but not required)

## After Completing Tutorials

Once you've completed the tutorials, you can:

- Explore [How-to Guides](../how_to/README.md) for specific tasks
- Read [Explanations](../explanations/README.md) to deepen your understanding
- Reference [API Documentation](../reference/README.md) for details
